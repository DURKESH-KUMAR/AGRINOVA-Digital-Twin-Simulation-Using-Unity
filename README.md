# 🌱 AGRINOVA – Digital Twin Simulation using Unity

## 📌 Overview
AGRINOVA is a Digital Twin-based simulation system developed using Unity to monitor, analyze, and optimize agricultural storage conditions—specifically focused on onion preservation.

This project simulates environmental conditions such as:
- Temperature  
- Humidity  
- Gas Levels (H₂S)

The system helps in predicting onion spoilage, sprouting time, and storage health status.

---

## 🚀 Features
- Digital Twin Simulation of agricultural storage
- Real-time sensor data integration (CSV / IoT-based)
- Smart Alerts System (OK / Caution / Critical)
- Spoilage & Sprouting Prediction
- AI-based Advisory System
- Interactive Unity UI

---

## 🧩 System Architecture
Real-world Data → Processing → Unity Simulation → AI Prediction → Output

## 🧪 How It Works
1. Load sensor data
2. Process temperature, humidity, gas values
3. AI evaluates conditions
4. Predict spoilage & sprouting
5. Output status and suggestions

---

## 📏 Constraints Used
- Temperature: 25°C – 30°C
- Humidity: 65% – 75%
- Gas Level: >250 indicates spoilage

Logic:
If temperature/humidity goes out of range → spoilage → gas increases
